# Nebula 🎵

A clean, minimal music player Android app for your local music collection.

## Features
- 🎵 Play pre-loaded local songs
- ⏯️ Play / Pause control
- ⏭️ Next / Previous track
- 🎚️ Seek slider to jump to any part of a song
- 🌙 Dark themed UI

## Tech Stack
- Kotlin
- MediaPlayer API
- ViewBinding

## How to Run
1. Clone the repo
2. Open in Android Studio
3. Add your MP3 files to `res/raw/` folder
4. Run on emulator or device

> ⚠️ MP3 files are excluded from this repo via `.gitignore`  
> You need to add your own songs locally to hear audio
